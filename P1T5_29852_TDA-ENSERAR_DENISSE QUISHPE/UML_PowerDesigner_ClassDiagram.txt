Pasos sugeridos para exportar a C++ en PowerDesigner:
1. Crear nuevo Data Model (Physical Data Model) o UML Model (dependiendo de la versión).
2. Añadir un Diagrama de Clases y crear las clases con atributos y métodos.
3. En el menú Model, usar la opción de generación de código (Model -> Generate Code) y seleccionar C++.
4. Configurar plantillas de generación si deseas nombres de ficheros concretos.
5. Exportar y copiar los archivos .h/.cpp generados a tu proyecto en Visual Studio Code.


Fichas de diseño (resumen):
- Asegúrate que 'enserar' es un puntero a float que se asigna con new float[filas*columnas*3] y se libera con delete[].
- Las matrices 2D (matriz1/2/3) deben ser creadas como float** con memoria dinámica: matriz[i] = new float[columnas]; y después liberar cada fila con delete[] y luego delete[] matriz.
- IngresarDatos debe pedir los números al usuario mediante cin.
- No es necesario retornar enserar; se mantiene como miembro de la clase.
